# Features Overview

## Core Features

### 1. Text Annotation
- Click anywhere on the PDF to add text
- Fully customizable font size (8-72px)
- Color picker for text color
- Drag and reposition text anywhere
- Delete text annotations
- Support for multi-line text

### 2. Signature Drawing
- Draw signatures with mouse or touch
- Smooth, natural drawing experience
- Clear and redraw functionality
- Drag and resize signatures
- Professional appearance

### 3. PDF Management
- Upload any PDF file
- Multi-page support with navigation
- High-quality rendering (1.5x scale)
- Download edited PDFs
- Preserves original PDF quality

### 4. User Interface
- Modern, gradient design
- Responsive layout (mobile-friendly)
- Intuitive toolbar
- Modal signature dialog
- Empty state with helpful instructions

## Technical Features

### Privacy & Security
- 100% client-side processing
- No data sent to any server
- No tracking or analytics
- Files never leave your device
- No account or login required

### Browser Compatibility
- Chrome/Chromium browsers
- Firefox
- Safari
- Edge
- Opera

### Deployment Options

#### 1. Standalone Web Application
- Single HTML file with CSS and JavaScript
- Can be hosted on any static server
- Works offline after initial load
- No build process required

#### 2. Chrome Extension
- Accessible from browser toolbar
- Can edit PDFs opened in Chrome
- Popup interface
- Separate editor tab
- Detect PDFs on web pages

#### 3. Embeddable Widget
- Inline embedding in existing pages
- Modal/popup mode
- Configurable size and appearance
- Simple JavaScript API
- No framework dependencies

## Limitations

### Current Limitations
- Cannot edit existing text in PDF
- Cannot remove existing PDF content
- Signature is raster (PNG) not vector
- Limited to standard fonts
- No OCR functionality
- No form field detection

### Future Enhancement Ideas
- More drawing tools (shapes, arrows)
- PDF page reordering
- Multiple signature presets
- Image insertion
- Highlight and markup tools
- Template system
- Cloud storage integration (optional)
